# TuringTest for LLMs - 项目状态报告

## 🎯 项目概述
TuringTest for LLMs 是一个利用原始设备模拟图灵测试的智能检测工具，通过与大型语言模型（LLM）的互动，评估模型是否展现出类似人类的智能行为。

./start.sh
